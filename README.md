##OneWord --- aka Yiju.us a personal project by Viking and FromSummer Team
*the source code of this project is now open-source, still under development,stay tuned!*
A [Viking] [1] project


Frameworks used in this project

* [Sea.js] [2] + [Backbone.js] [3] + [jQuery] [4] as javascript frameworks
* [Normalize.css] [5] + [HTML5 Boilerplate] [6] + [LESS] [7] for writing CSS + HTML 
* PHP [Slim] [8] framework to provide a RESTful api for frontend

to be continued...really wanna make it big
